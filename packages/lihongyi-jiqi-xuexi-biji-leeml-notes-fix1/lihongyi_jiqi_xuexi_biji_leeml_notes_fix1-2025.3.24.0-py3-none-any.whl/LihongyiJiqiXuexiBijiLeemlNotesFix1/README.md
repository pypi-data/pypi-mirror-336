## 下载

### Docker

```
docker pull apachecn0/lihongyi-jiqi-xuexi-biji-leeml-notes
docker run -tid -p <port>:80 apachecn0/lihongyi-jiqi-xuexi-biji-leeml-notes
# 访问 http://localhost:{port} 查看文档
```
