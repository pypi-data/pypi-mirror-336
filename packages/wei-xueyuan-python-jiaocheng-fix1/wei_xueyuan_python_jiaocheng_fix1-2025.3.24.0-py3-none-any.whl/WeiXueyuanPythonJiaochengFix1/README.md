# 微学苑 Python 教程

```
docker pull apachecn0/wei-xueyuan-python-jiaocheng
docker run -tid -p <port>:80 apachecn0/wei-xueyuan-python-jiaocheng
# 访问 http://localhost:{port} 查看文档
```
