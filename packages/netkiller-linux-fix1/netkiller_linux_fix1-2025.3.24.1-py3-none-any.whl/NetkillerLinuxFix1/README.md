# Netkiller-Linux

## 下载

### Docker

```
docker pull apachecn0/netkiller-linux
docker run -tid -p <port>:80 apachecn0/netkiller-linux
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install netkiller-linux
netkiller-linux <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g netkiller-linux
netkiller-linux <port>
# 访问 http://localhost:{port} 查看文档
```
