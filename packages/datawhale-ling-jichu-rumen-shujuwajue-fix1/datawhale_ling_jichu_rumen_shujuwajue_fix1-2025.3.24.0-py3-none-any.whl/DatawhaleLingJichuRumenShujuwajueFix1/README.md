# Datawhale 零基础入门数据挖掘

## 下载

### Docker

```
docker pull apachecn0/datawhale-ling-jichu-rumen-shujuwajue
docker run -tid -p <port>:80 apachecn0/datawhale-ling-jichu-rumen-shujuwajue
# 访问 http://localhost:{port} 查看文档
```
