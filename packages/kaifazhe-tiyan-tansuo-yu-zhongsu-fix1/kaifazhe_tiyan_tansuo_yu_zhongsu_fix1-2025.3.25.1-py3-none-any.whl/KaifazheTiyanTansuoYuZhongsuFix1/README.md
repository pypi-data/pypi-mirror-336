# 开发者体验：探索与重塑

## 下载

### Docker

```
docker pull apachecn0/kaifazhe-tiyan-tansuo-yu-zhongsu
docker run -tid -p <port>:80 apachecn0/kaifazhe-tiyan-tansuo-yu-zhongsu
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```

# 访问 http://localhost:{port} 查看文档
```
