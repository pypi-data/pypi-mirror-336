# Linux 命令行与 Shell 脚本教程

## 下载

### Docker

```
docker pull apachecn0/linux-minglingxing-yu-shell-jiaoben-jiaocheng
docker run -tid -p <port>:80 apachecn0/linux-minglingxing-yu-shell-jiaoben-jiaocheng
# 访问 http://localhost:{port} 查看文档
```
