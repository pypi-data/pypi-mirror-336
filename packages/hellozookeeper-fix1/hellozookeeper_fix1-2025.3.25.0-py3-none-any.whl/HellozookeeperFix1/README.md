# HelloZooKeeper

## 下载

### Docker

```
docker pull apachecn0/hellozookeeper
docker run -tid -p <port>:80 apachecn0/hellozookeeper
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install hellozookeeper
hellozookeeper <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g hellozookeeper
hellozookeeper <port>
# 访问 http://localhost:{port} 查看文档
```
