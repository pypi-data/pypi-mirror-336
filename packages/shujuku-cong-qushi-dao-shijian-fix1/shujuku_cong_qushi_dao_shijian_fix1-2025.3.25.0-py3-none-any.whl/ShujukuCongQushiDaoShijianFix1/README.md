## 下载

### Docker

```
docker pull apachecn0/shujuku-cong-qushi-dao-shijian
docker run -tid -p <port>:80 apachecn0/shujuku-cong-qushi-dao-shijian
# 访问 http://localhost:{port} 查看文档
```

```

# 访问 http://localhost:{port} 查看文档
```
