### Docker

```
docker pull apachecn0/shang-guigu-java-jishu-zhi-javaweb
docker run -tid -p <port>:80 apachecn0/shang-guigu-java-jishu-zhi-javaweb
# 访问 http://localhost:{port} 查看文档
```
