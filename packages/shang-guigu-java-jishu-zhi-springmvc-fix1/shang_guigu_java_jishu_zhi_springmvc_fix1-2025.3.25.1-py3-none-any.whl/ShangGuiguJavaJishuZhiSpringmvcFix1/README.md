# 尚硅谷 Java 技术之 SpringMVC

## 下载

### Docker

```
docker pull apachecn0/shang-guigu-java-jishu-zhi-springmvc
docker run -tid -p <port>:80 apachecn0/shang-guigu-java-jishu-zhi-springmvc
# 访问 http://localhost:{port} 查看文档
```

### PYPI

