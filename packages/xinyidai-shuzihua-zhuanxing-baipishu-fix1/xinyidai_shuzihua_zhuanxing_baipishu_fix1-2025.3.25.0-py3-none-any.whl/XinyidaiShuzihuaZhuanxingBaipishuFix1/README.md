## 下载

### Docker

```
docker pull apachecn0/xinyidai-shuzihua-zhuanxing-baipishu
docker run -tid -p <port>:80 apachecn0/xinyidai-shuzihua-zhuanxing-baipishu
# 访问 http://localhost:{port} 查看文档
```
