# 千古前端图文教程

## 下载

### Docker

```
docker pull apachecn0/qiangu-qianduan-tuwen-jiaocheng
docker run -tid -p <port>:80 apachecn0/qiangu-qianduan-tuwen-jiaocheng
# 访问 http://localhost:{port} 查看文档
```

