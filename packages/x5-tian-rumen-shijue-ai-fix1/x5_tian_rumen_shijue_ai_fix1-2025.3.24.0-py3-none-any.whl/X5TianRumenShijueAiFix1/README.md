## 下载

### Docker

```
docker pull apachecn0/x5-tian-rumen-shijue-ai
docker run -tid -p <port>:80 apachecn0/x5-tian-rumen-shijue-ai
# 访问 http://localhost:{port} 查看文档
```

