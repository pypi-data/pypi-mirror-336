# 人件 3rd

## 下载

### Docker

```
docker pull apachecn0/renjian-3rd
docker run -tid -p <port>:80 apachecn0/renjian-3rd
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install renjian-3rd
renjian-3rd <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g renjian-3rd
renjian-3rd <port>
# 访问 http://localhost:{port} 查看文档
```
