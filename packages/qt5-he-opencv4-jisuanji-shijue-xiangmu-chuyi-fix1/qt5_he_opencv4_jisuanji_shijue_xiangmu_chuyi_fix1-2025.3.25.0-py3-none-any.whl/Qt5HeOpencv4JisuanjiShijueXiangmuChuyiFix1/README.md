# Qt5 和 OpenCV4 计算机视觉项目（初译）

## 下载

### Docker

```
docker pull apachecn0/qt5-he-opencv4-jisuanji-shijue-xiangmu-chuyi
docker run -tid -p <port>:80 apachecn0/qt5-he-opencv4-jisuanji-shijue-xiangmu-chuyi
# 访问 http://localhost:{port} 查看文档
```
