# 前端内参

## 下载

### Docker

```
docker pull apachecn0/qianduan-neican
docker run -tid -p <port>:80 apachecn0/qianduan-neican
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install qianduan-neican
qianduan-neican <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g qianduan-neican
qianduan-neican <port>
# 访问 http://localhost:{port} 查看文档
```
